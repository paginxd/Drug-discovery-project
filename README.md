## Project: Drug Discovery

Проект по анализу химических дескрипторов и прогнозу эффективности соединений против вируса гриппа.

Что делаем:
- EDA и очистка данных.
- Регрессия для IC50, CC50, SI.
- Классификация для 4 задач: IC50 > median, CC50 > median, SI > median, SI > 8.
- Сравнение моделей и выбор лучших решений.

Структура:
- `data/` - исходные и очищенные данные (`df_clean.csv`).
- `notebooks/` - все ноутбуки по этапам: EDA и 7 модельных задач.
- `reports/` - итоговые таблицы метрик:
    - `regression_summary.csv`
    - `classification_summary.csv`
- `Аналитический отчет.docx` - подробный аналитический отчет по проделанной работе, результатам и выводам.

Итог проекта:
- Получены и сравнены результаты всех моделей по каждой задаче.
- Отдельно зафиксированы лучшие модели и метрики.
- В отчете объяснено, почему выбранные модели показали лучший результат с учетом типа данных.
